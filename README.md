# Build Aura: Music and Podcast API

### Project Description

You are going to build Aura, a streaming metadata API for podcasts and music. You will start with the secure Authify authentication template. Then, you will build user profiles with distinct roles: Listeners and Creators. You will learn how to handle raw file uploads by sending avatar images to Cloudinary and saving their URLs in your database. You will write strict, role based permissions so only Creators can upload podcasts. Finally, you will unlock advanced Django features by using database aggregation to calculate average podcast ratings instantly on the fly.

## Tech Stack

- Django
- Django REST Framework
- Django REST Framework SimpleJWT
